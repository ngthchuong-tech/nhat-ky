Gemini API key (tùy chọn — để viết kết luận bằng AI)
Không bắt buộc. Nếu để trống, nút "Tự động tổng hợp" ở mục Kết luận vẫn hoạt động bình thường (không cần AI). Nếu bạn dán API key Gemini của mình vào đây, sẽ có thêm nút "Viết lại bằng AI" để diễn đạt kết luận mượt hơn. Key chỉ lưu trên trình duyệt này (không gửi cho ai khác ngoài Google) — nên giới hạn key theo tên miền (HTTP referrer) trong Google AI Studio để an toàn hơn.

Điền các trường bên dưới. Bấm "+ Thêm" ở mỗi mục để bổ sung dòng/đối tượng nếu cần. Khi xong, bấm "Xuất file Word" ở cuối trang.
